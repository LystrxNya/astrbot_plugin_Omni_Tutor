2. 曲线的弧长

(1) $s = \int_{a}^{b} \sqrt{1 + y'^2} dx = \int_{a}^{b} \sqrt{1 + f'^2(x)} dx$, $y = f(x)$ $(a \leq x \leq b)$

(2) $s = \int_{\alpha}^{\beta} \sqrt{\phi'^2(t) + \psi'^2(t)} dt$, $\begin{cases} x = \phi(t) \\ y = \psi(t) \end{cases}$ $(\alpha \leq t \leq \beta)$

(3) $s = \int_{\alpha}^{\beta} \sqrt{r^2(\theta) + r'^2(\theta)} d\theta$, $r = r(\theta)$ $(\alpha \leq \theta \leq \beta)$

3. 已知平行截面面积函数为 $A(x)$ 的立体体积：$V = \int_{a}^{b} A(x) dx$
